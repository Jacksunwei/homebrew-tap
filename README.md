# homebrew-tap

Homebrew tap for jacksunwei's tools.

## Install

```bash
brew tap jacksunwei/tap
```

## Install Packages

```bash
# Casks
brew install --cask jacksunwei/tap/tapsense
```

### Casks

*   **tapsense** — MacBook accelerometer-based gesture triggers.

## Update / Uninstall

```bash
brew update
brew upgrade tapsense
brew uninstall --cask tapsense
```
